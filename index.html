// OVERVIEW

What This API Does

A production-grade cloud backend with a clean layered architecture (routers → services → models).

🔐

JWT Authentication

Register and login to receive a Bearer token. All write operations and order endpoints are protected.

📦

Product Management

Full CRUD for products with name, description, price, and real-time stock tracking.

🛒

Order Processing

Place multi-item orders with atomic stock deduction and validation. Update status, cancel orders.

✅

Input Validation

All inputs validated with Pydantic. Returns 400/404/422 with clear error messages.

📄

OpenAPI / Swagger

Full Swagger UI at /docs with live testing and schema browser.

☁️

Cloud Deployed

Static front-end hosted on AWS S3 with public access policy and static website hosting.

How to Use the API

Get started in 3 steps using Swagger UI at /docs.

01

Register an Account

POST your email and password to /auth/register. An account will be created and returned.

POST /auth/register → { email, password }
02

Login and Get Your Token

POST your credentials to /auth/login. Copy the access_token from the response.

POST /auth/login → { access_token, token_type }
03

Authorize and Call Protected Endpoints

Click the Authorize 🔓 button in Swagger, paste your token, then freely call all product and order endpoints.

Bearer {access_token} → unlocks 🔒 endpoints
